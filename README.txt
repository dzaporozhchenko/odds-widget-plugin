This plugin allows you to output odds data for specified games.

== Description ==

Plugin uses the [the-odds-api](https://the-odds-api.com/). You need to get an API key. Free tier will give you 500 requests per month.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place widget to your templates with the block editor.
